# Network Configuration & Troubleshooting
# Description
A simulated network environment built using Cisco Packet Tracer, demonstrating LAN topology design, router and switch configuration, and network troubleshooting using Cisco IOS commands.
# Tools Used
Cisco Packet Tracer, Cisco IOS, Cisco 2901/2911 Routers, Cisco 2960 Switches, and Cisco PC's

# What I Configured
Built and configured LAN topologies spanning 10+ devices including routers, switches and PCs. Used show commands including show ip route, show interfaces, show arp and show flash to monitor and troubleshoot network performance.

# What I Learned
Designing and implementing LAN topologies in a simulated environment using Cisco IOS CLI commands to configure and troubleshoot network devices 
and how to diagnose and resolve connectivity issues to maintain stable network performance.
