# Network Configuration & Troubleshooting

A simulated enterprise network environment built to demonstrate LAN topology design, hardware configuration, and infrastructure troubleshooting methodologies.

**Note: You can view screenshots of my project by clicking on Command Outputs.png and Network topology.png.**

## Technical Environment

- **Simulation Tools:** Cisco Packet Tracer
- **Hardware Profile:** Cisco 2901/2911 Routers, Cisco 2960 Switches, Enterprise Endpoints (PCs/Laptops)
- **Operating System:** Cisco IOS (Command Line Interface)

---

## Infrastructure Implementations

- **Topology Architecture:** Designed and deployed scalable Local Area Network (LAN) topologies bridging 10+ active infrastructure devices.
- **Device Provisioning:** Configured local host parameters, VLAN assignments, and routing boundaries via the Cisco IOS CLI.
- **Diagnostics & Monitoring:** Utilized advanced administrative commands to audit and verify network states:
  - `show ip route` — Analyzed routing tables and static/dynamic paths.
  - `show interfaces` — Monitored interface operational status and error rates.
  - `show arp` — Verified IP to MAC address resolutions across segments.
  - `show flash` — Inspected system memory and IOS image integrity.

---

## Technical Competencies Demonstrated

- **Network Design:** Practical implementation of foundational networking concepts including subnet mapping and interface configuration.
- **System Diagnostics:** Proven ability to logically isolate and resolve link failures, addressing mismatches, and routing loops to ensure stable network uptime.
